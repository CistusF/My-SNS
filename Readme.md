# My SNS
Easy to share your SNS
------------------------------

> How to use?  

edit `script.js`'s configuration

```js
const username = "Your name"; // Enter your name or nickname
const SNS_List = [
    {
        link: "https://instagram.com/cistusf",
        type: "instagram",
        alt: "My public IG"
    },
    {
        link: "https://youtube.com/@cistusf",
        type: "youtube"
    }
]; // Add your SNS list
```
